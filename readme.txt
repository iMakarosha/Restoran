Нужно восстановить бд из файла restoran.bak, обязательно переименовать. Добавила и другие файлы, но их лучше не использовать. 
После восстановления БД изменить строку подключения к БД можно в файле app.config; после изменения файла app.config необходимо открыть свойства проекта Restoran (не решения, проекта), перейти на вкладку "Параметны" и в предложенном диалоге подтвердить изменения, после чего сохранить файл. Если диалогового окна не было, нужно вручную заменить значение переменной RestoranConnectionString на новое.
